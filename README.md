# Classroom Reading Assistant

一个专注于课堂阅读辅助的交互式网页应用，旨在提升语言学习中的阅读与听力练习体验。

## 核心功能

- **双语阅读支持**：提供中英文对照文章，一键切换语言（快捷键：Z）
- **智能朗读**：支持句子/单词级别的朗读模式（快捷键：X切换），带实时高亮效果
- **个性化设置**：可调节语速、字号，选择不同朗读声音
- **便捷导航**：文章目录快速切换，首句/上一句/下一句快速跳转（支持键盘快捷键）
- **沉浸式体验**：支持显示/隐藏课文内容，专注听力练习（快捷键：C）

## 适用场景

- 学生英语/语文阅读能力提升
- 课堂互动教学辅助工具
- 听力理解训练
- 发音模仿练习

## 技术栈

- HTML5 + CSS3（响应式设计，适配多种设备）
- JavaScript（原生语音合成API，交互逻辑）
- 语义化UI设计，注重可访问性

## 使用说明

1. 点击目录按钮（M键）选择文章
2. 空格键/回车键播放下一句，退格键播放上一句
3. 调整设置面板中的语速、字号以获得最佳体验
4. 按P键查看完整快捷键指南

欢迎体验并提出改进建议！
